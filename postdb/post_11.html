$this->db->query()

To submit a query, use the following function:

1$this->db->query('YOUR QUERY HERE');

place raw SQL in the query() method

1$query = $this->db->query("select * from tbl_user");

Pass SQL query in a variable

1$sql = "select * from tbl_user";
2$query = $this->db->query($sql);

$this->db->query() With Query Bindings

Note:Use Of Query Bindings
Benefit of using binds is that the values are automatically escaped, producing safer queries

1$sql = "SELECT * FROM tbl_user WHERE name = ? AND type = ?";
2$this->db->query($sql, array('code', 'php'));
